# example-webdriverio
Example of how to integrate Percy visual tests with WebdriverIO.

## Running the example

```bash
$ npm install
$ PERCY_TOKEN=... npm test

$ git clone https://github.com/BrowserStackCE/percy-webdriverio-bstack-demo
$ cd percy-webdriverio-bstack-demo

$ export PERCY_TOKEN=...
$ npm install
$ npm test
```
